# 时间时区子系统

## 简介

 时间时区为OpenHarmony系统提供了管理系统时间时区和定时的能力，包括：

-   **管理时间时区**

    统一管理系统时间时区，包括设置/获取系统时间、日期、时区，同时提供获取系统启动时间。

-   **定时能力**

    提供系统定时器能力。包括定时器创建、启动、停止和销毁。定时器类型提供三种：系统启动时间计时类定时器、系统当前时间计时类定时器、唤醒类定时器。
